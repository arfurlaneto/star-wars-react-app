# Star Wars React App
Simple react app using Star Wars API (https://swapi.dev/) for onboarding of new employees.

# Instructions
1. Create the app.
```bash
npx create-react-app star-wars-react-app --template typescript
```

2. Install axios.
```bash
yarn add axios
```

2. Run the app.
```
yarn run start
```

3. Follow the commits.
