# Star Wars React App
Simple react app using Star Wars API (https://swapi.dev/) for onboarding of new employees.

# Instructions
1. Create the app.
```bash
npx create-react-app star-wars-react-app --template typescript
```
2. Open app folder.
```bash
cd star-wars-react-app
```

3. Install axios.
```bash
yarn add axios
```

4. Run the app.
```
yarn run start
```

5. Follow the commits.
